# 第1题——求素数

### 题目

> 输入一个数字，输出小于这个数字的所有素数。

### 举例

```
输入：20
输出：2 3 5 7 11 13 17 19
```

### 思路

* 思路：注意每次只比较到根号n就行。开平方函数是sqrt

### 复杂度

- 复杂度：（1-n）求和（n^(根号n)）


### 源码

```c++
int main() {
	//freopen("1.in","r",stdin);
	int n = 0;
	cin >> n;
	if (n < 2)
		return 0;
	cout << 2;
	for (int i = 3; i < n; i++) {
		int j = 2;
		int t = sqrt(i)+1;
		//cout << t << endl;
		for (j ; j < t; j++) {
			if (i%j == 0) {
				break;
			}
		}
		if (j == t) {
			cout << " " << i;
		}
	}
	system("pause");
	return 0;
}
```
